set_custom_title

set_custom_title(a_title)

新しく作るターミナルウィンドウに設定されるカスタムタイトルを設定します。

値が設定されると、新たに開かれるターミナルウィンドウのカスタムタイトルとして設定されます。設定しなければ、ターミナルでの設定が引き継がれます。

カスタムタイトル が設定されると、do, do_with はttyデバイス名とカスタムタイトルの両方が一致したターミナルを探します。見つからない場合は、同じカスタムタイトルのターミナルウィンドウを探します。

Parameters
  • a_title (text) : title for a terminal window
Result

script object : me
