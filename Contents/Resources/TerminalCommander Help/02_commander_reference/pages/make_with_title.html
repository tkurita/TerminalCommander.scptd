make_with_title

make_with_title(a_title)

ターミナルのタイトルを設定してインスタンスを生成します。

このメソッドは次のように、make でインスタンスを生成して、set_custom_title を呼ぶのと同じです。

(make TerminalCommander)'s set_custom_title(a_title)

Parameters
  • a_title (Unicode text) : ターミナルのタイトル
Result

script object : 新しいインスタンス
