TerminalCommander Reference
resolve_terminal

resolve_terminal given allowing_busy:allow_busy_flag

do_command もしくは do_in_new_term で開いたターミナルウィンドウの存在を確認します。もし存在しなければ、カスタムタイトルでウィンドウを探します。

Parameters
  • allowing_busy (boolean) : ターミナルウィンドウの busy プロパティが true （ターミナルで何らかのコマンドが実行中であるかどうか）であることを許容するかどうか。true を与えると、見つけたターミナルでコマンドが実行できるとは限りません。
Result

boolean : ターミナルが見つかったら true
