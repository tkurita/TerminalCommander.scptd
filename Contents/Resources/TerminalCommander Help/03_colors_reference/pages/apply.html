TerminalColors Reference Edit Script
apply

apply(a_commander)

ターミナルウィンドウのカラー設定をまとめて適用します。

apply_normal_text、apply_bold_text、apply_cursor、apply_background をまとめて実行しますが、それらを個別に実行するのとは違い、ターミナルウィンドウの確認を行います。

Parameters
  • a_commander : ターミナルウィンドウへの参照を保持している TerminalCommander のインスタンス
Result

boolean : 成功したら true。a_commander に与えた TerminalCommander のインスタンスがターミナルウィンドウを発見できなかったら false。
