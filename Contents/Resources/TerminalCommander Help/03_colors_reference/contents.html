TerminalColors Reference

  • Version : 2.1
  • Author : Tetsuro KURITA tkurita@mac.com

TerminalColors は TerminalCommander で操作するターミナルの色を設定します。詳しい使い方は、TerminalCommander のマニュアルを参照してください。

Constructors

make

TerminalColors のインスタンスを生成します。

Setting Terminal Colors

色指定には、0 から 65535 の範囲の、3つの数値からなるリストを与えます。それぞれの成分がRGBに対応します。

  • {65535, 65535, 65535} : 白
  • {0, 0 , 0} : 黒
  • {65535, 0, 0} : 赤
  • {0, 65535, 0} : 緑
  • {0, 0, 65535} : 青

色を設定するメソッドには、色指定の他に実際にその値を適用するかどうかを引数に与えます。　false を設定すると、色設定は適用されません。つまり、色設定を覚えてはおくが、実際には使わないということができます。

set_normal_text

通常のテキスト色を設定します。

normal_text

設定されている通常テキストのカラーを取得します。

set_change_normal_text

set_normal_text で設定した色設定を適用するかどうかを設定します。

is_change_normal_text

通常テキストカラーの設定を適用する設定になっているかどうか調べます。

set_bold_text

ボールドテキストのカラー設定を行います。

bold_text

設定されているボールドテキストのカラーを取得します。ボールドテキストカラーは set_bold_text によって設定されます。

set_change_bold_text

set_bold_text で設定したカラー設定を適用するかどうかを設定します。

is_change_bold_text

ボールドテキストカラーの設定を適用する設定になっているかどうか調べます。

set_cursor

カーソルカラーを設定します。

cursor_color

設定されているカーソルカラーを取得します。

set_change_cursor

set_cursor で設定したカラー設定を適用するかどうかを設定します。

is_change_cursor

カーソルカラーの設定を適用する設定になっているかどうか調べます。

set_background

ウィンドウの背景のカラーおよび透明度の設定を行います。

background_color

設定されているウィンドウ背景のカラーを取得します。

background_opaqueness

設定されているウィンドウの透明度を取得します。ウィンドウの透明度は set_background によって設定されます。

set_change_background

set_background で設定した色設定を適用するかどうかを設定します。

is_change_background

ウィンドウ背景のカラー設定を適用する設定になっているかどうか調べます。

Applying Colors

apply

ターミナルウィンドウのカラー設定をまとめて適用します。

apply_normal_text

set_normal_text で設定した通常テキストのカラー設定を既に開かれているターミナルウィンドウに適用します。

apply_bold_text

set_bold_text で設定したボールドテキストのカラー設定を既に開かれているターミナルウィンドウに適用します。実行する前に、TerminalCommander によって、ターミナルウィンドウが開かれている必要があります。

apply_cursor

set_cursor で設定したカーソルのカラー設定を既に開かれているターミナルウィンドウに適用します。実行する前に、TerminalCommander によって、ターミナルウィンドウが開かれている必要があります。

apply_background

set_background で設定したウィンドウ背景のカラー設定を既に開かれているターミナルウィンドウに適用します。
