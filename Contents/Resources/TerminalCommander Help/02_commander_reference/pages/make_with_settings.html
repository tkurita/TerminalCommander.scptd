make_with_settings

make_with_settings(a_name)

ターミナルの設定セットを指定してインスタンスを生成します。

このメソッドは次のように、make でインスタンスを生成して、set_settings_name を呼ぶのと同じです。

(make TerminalCommander)'s set_settings_name(a_name)

Parameters
  • a_name (Unicode text) : 設定セット名
Result

script object : 新しいインスタンス
