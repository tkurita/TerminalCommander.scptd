apply_normal_text

apply_normal_text(term_window)

set_normal_text で設定した通常テキストのカラー設定を既に開かれているターミナルウィンドウに適用します。

Parameters
  • term_window (reference) : ターミナルウィンドウ
Result

: 通常テキストのカラー設定の適用に成功したら true。set_normal_text のa_bool に false が与えられていたり、そもそもカラー設定が行われていない場合は false。
