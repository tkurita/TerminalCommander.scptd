apply_cursor

apply_cursor(term_window)

set_cursor で設定したカーソルのカラー設定を既に開かれているターミナルウィンドウに適用します。実行する前に、TerminalCommander によって、ターミナルウィンドウが開かれている必要があります。

Parameters
  • term_window (reference) : ターミナルウィンドウ
Result

boolean : カーソルのカラー設定の適用に成功したら true。set_cursor のa_bool に false が与えられていたり、そもそもカラー設定が行われていない場合は false。
