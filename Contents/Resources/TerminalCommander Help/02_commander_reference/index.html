TerminalCommander Reference

  • Version : 2.2.1
  • Author : Tetsuro KURITA tkurita@mac.com

Constructors

make

新しいインスタンスを生成します。

make_with_title

ターミナルのタイトルを設定してインスタンスを生成します。

make_with_settings

ターミナルの設定セットを指定してインスタンスを生成します。

Sending Commands to Terminal

do_command

ターミナルでシェルコマンドを実行します。必要に応じて、新しいターミナルウィンドウを開きます。

do_in_new_term

新しいターミナルを開いてシェルコマンドを実行します。

wait_termination

do_command で実行したコマンドの終了を待ちます。

resolve_terminal

do_command もしくは do_in_new_term で開いたターミナルウィンドウの存在を確認します。もし存在しなければ、カスタムタイトルでウィンドウを探します。

Terminal Properties Setting

ターミナルのタイトル、エンコーディング、シェルに関する設定は、 do_command を実行する前に以下の method を使ってインスタンスに設定しておく必要があります。以下の method によって設定される属性は、新しいターミナルが作られるときだけ適用されます。

set_settings_name

新規ターミナルで使用する設定セット名を設定します。

set_colors

ターミナルの色を設定します。TerminalColors のインスタンスを与えます。

set_custom_title

新しく作るターミナルウィンドウに設定されるカスタムタイトルを設定します。値が設定されると、新たに開かれるターミナルウィンドウのカスタムタイトルとして設定されます。設定しなければ、ターミナルでの設定が引き継がれます。また、カスタムタイトル が設定されると、do_command はターミナルの参照が見つからないときは、同じ カスタムタイトル のターミナルウィンドウを探します。

set_custom_title_for_path

新しく作るターミナルウィンドウに設定されるカスタムタイトルを与えられたパス文字列を使って設定します。パス文字列の3階層分がカスタムタイトルとして使われます。

custom_title

set_custom_title によって設定した、カスタムタイトルを返します。

set_use_osax_for_customtitle

ターミナルウインドウのカスタムタイトルの設定に TerminalControl.osax を使用します。Mac OS X 10.5-10.5.6, Mac OS X 10.6 でのタイトル設定に関するバグを回避します。

set_execution_string

新しいターミナルを開いたときだけ一番最初に実行するコマンドを設定します。シェルの環境設定のために実行したいコマンドを実行すると良いでしょう。

execution_string

set_execution_string によって設定された、新しいターミナルを開いたときだけ一番最初に実行するコマンドを設定します。

set_clean_commands

コマンドの入力を受け付けるプロセスの設定。シェルに加えて、ここで設定したプロセスがターミナルで起動していてもコマンドを送りつけられるにします。対話型プロセスを扱うときは設定が必要です。

clean_commands

set_clean_commands で設定した値を返します。

set_working_directory

Set a prefered working directory of a terminal window when finding a terminal.

set_visible_when_launching

ターミナルが起動していない時に、do_command でターミナルを起動した時に、ターミナル.app を表示するかどうか。

visible_when_launching

ターミナルが起動していない時に、do_command でターミナルを起動した時に、ターミナル.app を表示するかどうか。

Cheking Terminal Status

is_busy

ターミナルがコマンドを受け付けられる状態かどうか。set_clean_commands で設定したプロセス、およびデフォルトで設定されているプロセス以外が起動していたら true になります。

ttyname

保持しているターミナル参照の tty 名

window_contents

ターミナルで表示されている内容を取得します。

buffer_history

ターミナルウィンドウのスクロールバッファ内のすべての内容を取得します。

running_processes

参照しているターミナルで起動しているプロセスを取得します。

terminal_window

インスタンスが保持しているターミナルウィンドウ/タブへの参照

terminal_name

インスタンスが保持しているターミナルウィンドウの名前

Others

bring_to_front

保持しているターミナルウィンドウ を最前面に持ってきて、ターミナルを activate します。

check_osax

Check installation of TerminalControl.osax and its installed version.
