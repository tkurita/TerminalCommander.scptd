set_background

set_background(a_rgb, |opaque|, a_bool)

ウィンドウの背景のカラーおよび透明度の設定を行います。

新規ターミナルウィンドウを開くとき、もしくは apply_background 、apply が実行されるまで、設定はターミナルウィンドウに適用されません。

  • 0 -- 完全に透明
  • 65535 -- 完全に不透明
Parameters
  • a_rgb (list) -- RGB 色設定
  • opaque (integer) -- ターミナルウィンドウの透明度。0 から 65535 の範囲の数値を与えます。
  • a_bool -- false を設定すると、a_rgb は使われませんが、値は保持します。
Result

script object : me
